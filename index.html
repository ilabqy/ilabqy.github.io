This data set consists of a set of 37 stereo image pairs, together with three input images for each: the ground truth,the stroke and the disparity map. The purpose of the data set is to test selection algorithms on stereo image pairs.
The images consist of original images incllude ground truth（No.1-31）taken by the authors and images from the Adobe Stereo Selection Data Set[1]. The images in the Adobe Stereo Selection Data Set can be downloaded from http://www.adobe.com/go/datasets on the datasets page. The another six stereo image pairs together with the ground truth (No.32-37) selected from the dataset in [2].
